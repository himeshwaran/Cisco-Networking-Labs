# Cisco Packet Tracer - Console Connectivity Lab.

## 📘 Overview
This lab demonstrates how to access Cisco devices through console connectivity using Cisco Packet Tracer. The activity covers:

- 🔌 Connecting to a Cisco 2960 switch using a rollover console cable  
- 💻 Accessing Cisco IOS through terminal emulation  
- ⚙️ Exploring User EXEC and Privileged EXEC modes  
- ⏰ Configuring and verifying the system clock  
- 🔗 Connecting to a Cisco ISR4321 router using a mini-USB console cable  

This lab helped strengthen practical networking and Cisco IOS configuration skills.

---

## 🎯 Objectives

### Part 1: Access a Cisco Switch through the Serial Console Port
- Install and inspect a Cisco 2960 switch  
- Connect a PC to the switch using a rollover console cable  
- Access the Cisco IOS CLI through the terminal  

### Part 2: Display and Configure Basic Device Settings
- Display IOS version information  
- View and configure the system clock  
- Navigate between IOS modes  

### Part 3: Access a Cisco Router Using a Mini-USB Console Cable
- Install and inspect a Cisco ISR4321 router  
- Connect a laptop to the router using a mini-USB cable  
- Access the router CLI using terminal emulation  

---
