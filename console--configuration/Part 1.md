
# Part 1 - Access Switch Through Console Port

## Step 1: Add the Cisco 2960 Switch

1. Open Cisco Packet Tracer
2. Navigate to:

```text
Network Devices > Switches
```

3. Drag the Cisco 2960 switch to the workspace or rack.

---

## Step 2: Add the PC

1. Navigate to:

```text
End Devices > PC
```

2. Drag the PC to the workspace.

---

## Step 3: Connect the Console Cable

1. Click the Connections icon.
2. Select the blue Console Cable.
3. Connect:

```text
PC RS232 Port --> Switch Console Port
```

---

## Step 4: Open the Terminal

1. Click the PC
2. Open:

```text
Desktop > Terminal
```

3. Keep the default settings:

```text
9600 baud
8 data bits
No parity
1 stop bit
No flow control
```

4. Click OK.

5. Press ENTER when the following message appears:

```text
Press RETURN to get started!
```

You should now see:

```text
Switch>
```

This is User EXEC Mode.
