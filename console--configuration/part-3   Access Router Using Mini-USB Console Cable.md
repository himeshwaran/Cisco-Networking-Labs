# Part 3 - Access Router Using Mini-USB Console Cable

## Step 1: Add the Cisco ISR4321 Router

1. Navigate to:

```text
Network Devices > Routers
```

2. Drag the Cisco ISR4321 router to the workspace.
3. Turn ON the router using the power button.

---

## Step 2: Add the Laptop

1. Navigate to:

```text
End Devices > Laptop
```

2. Drag the laptop to the workspace.

---

## Step 3: Connect the Mini-USB Cable

1. Select the Mini-USB cable from Connections.
2. Connect:

```text
Laptop USB Port --> Router Mini-USB Console Port
```

---

## Step 4: Access Router CLI

1. Open the laptop.
2. Navigate to:

```text
Desktop > Terminal
```

3. Keep default settings and click OK.

After booting, the router displays:

```text
Would you like to enter the initial configuration dialog? [yes/no]:
```

Type:

```text
n
```

Press ENTER.

You should now see:

```text
Router>
```
